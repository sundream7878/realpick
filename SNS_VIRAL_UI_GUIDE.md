# SNS 바이럴 UI 사용 가이드

## 🎨 UI 구현 완료!

어드민 대시보드에서 직접 영상을 생성하고 확인할 수 있습니다.

## 📍 접근 방법

### 1. 리얼픽 마케터 대시보드 접속

http://localhost:5173

비밀번호: `realpick-admin-2024`

### 2. "SNS 바이럴 (영상 생성)" 탭 클릭

---

## 🎬 영상 생성 프로세스

### Step 1: 미션 선택
- 승인된 미션 목록에서 영상으로 만들 미션 클릭
- 미션 제목, 선택지 A/B 확인

### Step 2: Track 선택
3가지 스타일 중 선택:
- **🤖 AI 자동 미션**: 일반 사용자 타겟, 친근한 톤
- **🎬 딜러 파트너십**: 딜러 브랜딩 강조 (추후)
- **🔥 결과 중계**: 긴급 속보 스타일, 긴박감

### Step 3: 플랫폼 선택 (다중 선택 가능)
- ✅ Instagram (Reels)
- ✅ YouTube (Shorts)
- ✅ TikTok (추후)

### Step 4: 생성 시작
"AI 영상 생성 시작" 버튼 클릭

**진행 순서**:
```
1. Gemini AI 시나리오 생성 (10초)
   ↓
2. Canvas 영상 렌더링 (2~3분)
   ↓
3. SNS 콘텐츠 생성 (5초)
   ↓
4. 완료!
```

### Step 5: 결과 확인

#### 📹 생성된 영상
- 영상 다운로드 버튼 클릭
- 폴더 열기로 파일 위치 확인
- `temp/mission-xxx.mp4` 파일 확인

#### 📱 SNS 콘텐츠 (AI 생성)
각 플랫폼별로:
- **캡션**: 게시글 본문 (복사 가능)
- **해시태그**: #리얼픽 #프로그램명 + 하이재킹 태그
- **CTA**: 투표 유도 문구

#### 🎬 시나리오 JSON
"시나리오 JSON 보기" 버튼 클릭하면:
- Gemini가 생성한 전체 시나리오 확인
- 장면별 타이밍, 자막, 애니메이션 등

## 🎨 UI 기능 상세

### 미션 선택 영역
- 승인된 미션만 표시
- 선택 시 보라색 하이라이트
- 프로그램명 배지 표시

### Track 선택 영역
- 3개 카드 레이아웃
- 각 Track의 특징 설명

### 플랫폼 선택 영역
- 토글 가능 (다중 선택)
- 선택 시 아이콘 색상 변경

### 생성 진행 상황
- 로딩 인디케이터
- 예상 소요 시간 표시
- 단계별 안내 메시지

### 결과 화면
- **영상 정보**: 경로, 다운로드 버튼
- **SNS 콘텐츠**: 플랫폼별 카드
  - 캡션 (복사 가능)
  - 해시태그 (복사 가능)
  - CTA
- **시나리오 JSON**: 전체 시나리오 확인

## 🔄 워크플로우 (내부 동작)

```
[UI] 생성 버튼 클릭
    ↓
[API] POST /api/video/render
    ↓
[Gemini] 시나리오 생성
    - 10초 영상 구성
    - 장면별 자막, 타이밍
    - 애니메이션 효과
    ↓
[Canvas] 영상 렌더링
    - 300 프레임 생성 (10초 × 30fps)
    - 각 프레임을 PNG 이미지로 저장
    - FFmpeg로 MP4 합성
    ↓
[Gemini] SNS 콘텐츠 생성
    - Instagram 캡션 + 해시태그
    - YouTube 설명 + #Shorts
    - TikTok 본문 + 해시태그
    ↓
[Firestore] rendering_jobs 저장
    ↓
[UI] 결과 표시
```
