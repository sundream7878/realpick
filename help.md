# 현재 이슈 및 해결 현황 (2026-03-12)

## 1. 에펨코리아 로그인 및 권한 문제
- **증상**: 봇이 로그인을 성공했다고 판단했으나, 실제 글쓰기/댓글 작성 시 "권한이 없습니다" 또는 "로그인이 필요합니다" 오류 발생.
- **원인**: 
  1. **도메인 불일치**: 로그인(`www.fmkorea.com`)과 게시글 접근(`fmkorea.com`) 도메인이 달라 세션 쿠키가 공유되지 않음.
  2. **세션 만료**: 로그인 후 짧은 시간 내에 세션이 만료되거나 유지되지 않음.
- **조치 사항 (적용됨)**:
  - **URL 정규화**: 모든 접근 URL을 `www.fmkorea.com`으로 강제 통일.
  - **로그인 유지**: 로그인 시 '로그인 유지(Keep Signed In)' 체크박스를 자동으로 클릭하도록 수정.
  - **로그인 검증 강화**: URL 변경, 로그아웃 버튼 등장 여부, 에러 메시지 등을 3단계로 검증.
- **상태**: **해결 패치 적용 완료**. (지속적인 모니터링 필요)

## 2. 디시인사이드 댓글 등록
- **증상**: "Timed out receiving message from renderer" 오류 및 비회원 댓글 작성 실패.
- **조치 사항 (적용됨)**:
  - **페이지 로드 전략 변경**: `eager` 모드 적용으로 리소스 로딩 대기 시간 단축.
  - **비회원 로직 추가**: 로그인이 풀렸을 경우 자동으로 닉네임/비밀번호를 입력하여 유동닉으로 작성 시도.
- **상태**: **완료**.

## 3. UI/UX 개선
- **증상**: '작성 완료 처리' 버튼의 글씨가 검은색이라 어두운 배경에서 잘 안 보임.
- **조치 사항 (적용됨)**: 버튼 텍스트 색상을 흰색(`text-white`)으로 변경.
- **상태**: **완료**.
